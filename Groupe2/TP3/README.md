# Bibliothèques
* stdio.h
* string.h (pour les chaînes de caractères)
* ctype.h (pour isdigit, isspace)
* stdlib.h (pour malloc, atoi)
* time.h (pour la commande date)

# Références
* Le support de cours (surtout pour les structures et les pointeurs)
* Quelques recherches sur internet pour le fonctionnement des lexers/parseurs

# Difficulté
* Ce TP était plus complexe que les précédents
* La partie lexer/parseur demandait de bien comprendre le découpage en tokens
* Les pointeurs et l'allocation dynamique étaient un peu piégeux au début
* Gérer les erreurs de syntaxe proprement n'était pas évident

# Commentaires
* On a appris comment fonctionne un interpréteur de commandes (REPL)
* L'ajout des commandes avec un tableau de structures était une bonne idée pour éviter les if/else à rallonge
* Le tokenizer nous a aidé à comprendre comment un compilateur analyse le code
* Beaucoup de temps passé sur la gestion des chaînes et des tokens
